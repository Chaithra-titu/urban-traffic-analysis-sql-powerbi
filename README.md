# urban-traffic-analysis-sql-powerbi
Urban traffic data analysis project using SQL, Excel, and Power BI. Identifies traffic patterns, congestion, accidents, and signal delays.

Tools & Technologies
1. Microsoft Power BI
2. Microsoft Excel
3. SQL (Structured Query Language) 


Workflow
1. Raw data collection
2. Data cleaning in Excel
3. SQL queries for analysis
4. Visualization using Power BI
5. Insight generation

Key Insights
1.  Industrial Zone: High accidents, high traffic (2393 vehicles), and high signal delay. This zone
requires the most attention.
2. Downtown Zone: High accidents (10) and highest signal delay with moderate traffic (~2100vehicles), Second priority area.
3. Airport Zone: Highest traffic (3190 vehicles) but relatively lower accidents and signal delay. Mainly
a high-traffic flow area.


Conclusion:
                 Industrial zone is the most critical area as it shows high values across all metrics. Downtown follows due to high delay and accidents, while Airport mainly handles high traffic volume.
